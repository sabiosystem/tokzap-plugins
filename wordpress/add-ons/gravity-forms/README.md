# TokZap — Gravity Forms Add-on

**Status:** 🚧 Em breve

Campo de verificação WhatsApp para formulários Gravity Forms. O usuário valida o telefone sem sair da página.

Quando disponível, será integrado via [API TokZap](https://tokzap.com/docs):

```
Base URL: https://api.tokzap.com/v1
Auth:     Authorization: Bearer tk_live_...
```

Acompanhe o lançamento em [tokzap.com](https://tokzap.com).
