# TokZap — n8n Community Node

**Status:** 🚧 Em breve

Node nativo para n8n que expõe as ações Send OTP, Verify OTP e Send Message da API TokZap em qualquer workflow.

Quando disponível, será integrado via [API TokZap](https://tokzap.com/docs):

```
Base URL: https://api.tokzap.com/v1
Auth:     Authorization: Bearer tk_live_...
```
